### Задание 1 Кеширование

Приведите примеры проблем, которые может решить кеширование.

Приведите ответ в свободной форме.

***Медленная загрузка страниц***

* **Проблема**:  Долгая загрузка сайта или приложения 
* **Решение**: Кэш хранит тяжелые элементы (картинки, стили, скрипты). Браузер или CDN отдает их мгновенно без повторной генерации

***Высокая нагрузка на базу данных***

* **Проблема**: Одинаковые сложные запросы к БД перегружают сервер.
* **Решение**: Результат первого запроса сохраняется в озу,следующие пользователи получают данные оттуда, разгружая БД.

***Дорогие и сложные вычисления***

**Проблема**: Сервер тратит много ресурсов процессора на расчеты
**Решение**: Результат расчета кэшируется. При повторном запросе сервер просто отдает готовое значение.
